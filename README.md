# Spot Version 3


**Spot Version 3** is a redesigned prototype of a student coursework management platform, created to improve usability, clarity, and overall student experience compared to the existing system.

The project focuses on clearer deadline tracking, better grade visibility, and more structured presentation of academic information, while exploring how thoughtful design and modern web technologies can improve everyday student workflows.
